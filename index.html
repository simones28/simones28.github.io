Simple Computer

One project I especially enjoyed working on was the creation of a simple computer in one of my classes. We used Verilog to code a function unit and write hexcodes and opcodes for instructions and turn those into machine code. We continued to use this simple computer and the memory set up to write assembly code that carried out an algorithm to take a set of numbers and return the maximum, minimum, and average values. I enjoyed using a computer at such a basic level and starting from 1s and 0s to end with a working algorithm.
